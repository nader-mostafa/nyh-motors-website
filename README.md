# NYH Motors Website

A modern car showcase website presenting luxury vehicles with a clean user interface and smooth browsing experience.

---

## Overview

NYH Motors is a front-end web project designed to display premium cars in a visually appealing and structured way.  
The project focuses on user experience, organized content, and a modern dark-themed design.

---

## Features

- Multi-page website structure  
- Featured cars and detailed car pages  
- Test drive booking page  
- Blog and articles section  
- Car comparison feature  
- Dealerships and contact pages  
- Login and registration system (UI)  
- Admin/Dashboard interface (UI)  
- Upcoming cars section  
- Responsive and clean layout  

---

## Technologies Used

- HTML5  
- CSS3  
- JavaScript
